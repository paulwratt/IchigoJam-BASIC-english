IchigoJam PC
-----------------------------------------------------------
[Function]
Return absolute value.

[Command format]
ABS(<value>)

[Example]
A = ABS(15)
I = ABS(A)

[Commentary]
When <value> is a negative value (minus)
Change it to a positive value (plus) and return it.
Values ​​greater than 0 will be returned without conversion.
