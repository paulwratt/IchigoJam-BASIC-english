[Function]
Generate a random number.

[Command format]
RND(<number>)

[Example]
RND(6)

[Commentary]
Returns an integer greater than or equal to 0 and less than <number>.
For example, if <number> is 6, it returns one from 0 to 5.
For 1.1.0 and later and 1.0.2 beta 11, initialization can be performed using SRND.
