[ 機能 ]
最新でアクセスしたファイル番号を得ます。

[ コマンド 書式 ]
FILE()

[ 例 ]
PRINT FILE()

[ 解説 ]
SAVE・LOAD・LRUN の最新でアクセスしたファイル番号を返します。
電源オンから一度ファイルアクセスを行っていない場合は -1 を返します。
